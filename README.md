# FX ALKILANY — Landing Page

صفحة هبوط بتوجّه تلقائيًا لقناة تليجرام `https://t.me/FXALKILANY`.

## التشغيل محليًا
```bash
npm install
npm run dev
```

## البناء
```bash
npm run build
```

## الرفع على Vercel
1. ارفع المجلد على GitHub (أو اسحبه مباشرة على Vercel).
2. على Vercel: سيتم قراءة `vercel.json` تلقائيًا.
3. Build command: `npm run build` — Output: `dist`.
4. مفيش أي متغيرات بيئة مطلوبة.
